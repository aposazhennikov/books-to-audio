# Review - Шпаргалка

## Быстрый старт

### 1. Проверить книгу (посмотреть проблемы)
```bash
python -m book_normalizer.cli process books/mybook.txt -v
# Найденные проблемы сохраняются в: data/user_memory/review_sessions/session_*.json
```

### 2. Исправить проблемы (интерактивно)
```bash
python -m book_normalizer.cli process books/mybook.txt --interactive
```

**Варианты выбора в интерактивном режиме**:
- `[1] Accept` - Принять предложенное исправление (система ЗАПОМНИТ!)
- `[2] Reject` - Отклонить (оставить как есть)
- `[3] Skip` - Пропустить (проверить позже)
- `[4] Edit` - Ввести свой вариант (система ЗАПОМНИТ!)
- `[5] Exit` - Выйти из проверки

### 3. Посмотреть память системы
```bash
# Память пунктуации
cat data/user_memory/punctuation_memory.json

# Память исправлений (OCR/орфография)
cat data/user_memory/correction_memory.json
```

## Как работает память

```
Первая книга          Вторая книга (автоматически!)
─────────────        ────────────────────────────
book1.txt            book2.txt
  ├─ Проблема: ,,      ├─ Проблема: ,, 
  ├─ Ты: Accept →      ├─ Система: AUTO-RESOLVED ✅
  │                    │
  ├─ Проблема: 0       ├─ Проблема: 0
  └─ Ты: Accept →      └─ Система: AUTO-RESOLVED ✅
```

## Отключить проверки

```bash
# Отключить пунктуацию
python -m book_normalizer.cli process books/mybook.txt --skip-punctuation

# Отключить орфографию
python -m book_normalizer.cli process books/mybook.txt --skip-spellcheck

# Отключить обе
python -m book_normalizer.cli process books/mybook.txt --skip-punctuation --skip-spellcheck
```

## Что обнаруживается

### Пунктуация (5 типов)
- `,,` → `,` (двойные запятые)
- `..` → `…` (два точки)
- `слово,слово` → `слово, слово` (нет пробела)
- `слово ,` → `слово,` (лишний пробел)
- `!!!` (чрезмерные знаки)

### OCR/орфография (3 типа)
- `вe` → `ве` (латиница в кириллице)
- `м0сква` → `москва` (цифры вместо букв: 0→О, 3→З, 6→б)
- `встрчстврнст` (подозрительные последовательности)

## Важно знать

✅ **Проверка ВСЕЙ книги** - система проверяет все главы за один проход
✅ **Последовательный просмотр** - проблемы показываются одна за другой
✅ **Автосохранение** - все решения сохраняются автоматически
✅ **Автоприменение** - знакомые проблемы исправляются без тебя

❌ **Не проверяет** - реальную орфографию ("превет"), грамматику ("в течении"), семантику
