# Tooling Preferences

- Prefer Context7 MCP for up-to-date library and framework documentation when it is available.
- Use Context7 before relying on memory for APIs, dependency behavior, install instructions, or version-sensitive examples.
- Prefer Playwright MCP for browser, web UI, screenshot, accessibility, and end-to-end interaction checks when it is available.
- Use Playwright MCP to verify UI behavior after meaningful frontend changes instead of relying only on static code inspection.
- If Context7 or Playwright MCP is unavailable in the current session, state that clearly and use the best available fallback.
